## üí° Tips for Success

### For Server Owners

1. **Read Installation Guide first**
   - Proper setup prevents issues
   - Check all dependencies

2. **Start with Quick Start Tutorial**
   - Create simple shop first
   - Add complexity gradually

3. **Use provided examples**
   - Copy from Shop-Examples.md
   - Modify for your needs

4. **Test thoroughly**
   - Test on staging server
   - Verify all conditions
   - Check console logs

5. **Keep backups**
   - Back up configurations
   - Document changes
   - Test before production

### For Developers

1. **Read API Documentation**
   - Understand architecture
   - Review code examples

2. **Study condition system**
   - Learn how checks work
   - Understand two-phase validation

3. **Check existing conditions**
   - Review implementations
   - Follow patterns

4. **Test thoroughly**
   - Unit tests
   - Integration tests
   - Edge cases

### Common Mistakes to Avoid

‚ùå **Don't:**
- Skip installation guide
- Copy configs without understanding
- Ignore error messages
- Use tabs in YAML
- Test on production first

‚úÖ **Do:**
- Read documentation
- Test incrementally
- Check console logs
- Use spaces in YAML
- Back up configs
